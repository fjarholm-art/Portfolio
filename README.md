# Fjarholm Portfolio

> A curated collection of independent digital artworks.

This repository contains the standalone portfolio presented at:

https://fjarholm.com/portfolio/

The collection brings together architectural concepts, vector artwork, and geometric pastel compositions developed through the Fjarholm visual language.

---

## Portfolio Collections

### Architecture

Minimalist architectural concepts inspired by Nordic landscapes, geometry, and contemporary design.

### Vector Outline Silhouettes

Clean SVG-based outline illustrations built through mathematical precision and scalable vector geometry.

### Pastel Art

Geometric compositions exploring atmosphere, nature, balance, and quiet seasonal transformation.

---

## Design Philosophy

Every artwork follows the same core principles:

- clarity over complexity
- geometry before decoration
- timeless visual balance
- digital craftsmanship inspired by nature
